# 모유수유 리얼 가이드 1.0 (Breastfeeding Real Guide)

## Overview
A static HTML website providing a comprehensive breastfeeding guide in Korean. Created by experts with 25 years of field experience and IBCLC professional knowledge.

## Project Structure
- `index.html` - Main single-page application containing all content and styles
- `assets/mg-chat.js` - JavaScript chat functionality
- `server.js` - Simple Node.js static file server for development

## Tech Stack
- Pure HTML/CSS/JavaScript (no build process required)
- Node.js static file server for development

## Running Locally
The project uses a simple Node.js server that serves static files on port 5000:
```
node server.js
```

## Deployment
Configured as a static site deployment, serving files from the root directory.
